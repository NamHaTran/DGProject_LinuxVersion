◆ checkPartitionedMesh()

bool preProcessKey::checkPartitionedMesh ( std::string  cmd)

Function return true if checkPartitionedMesh is available.

checkPartitionedMesh loops over all mesh cells to find isolated cells, then changes the master core to the lowest index master core of neighbour cell.
Explainations:

  • Isolated cell is the cell which has ALL neighbour cells belong to different master cores in comparison with itself.
  • Master core of cell is the core contains the cell after decomposing mesh.
