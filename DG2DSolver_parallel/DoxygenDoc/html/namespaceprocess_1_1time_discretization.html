void process::timeDiscretization::TVDRK_1step ( int  RKOrder)

Function solves 1 TVD_RK iteration.

Workflow explanation

Assume that U is at time n:

  • 1: Run Function Positivity preserving limiter for Rho (to calculate theta1) to keep Rho always positive while solving \(S_m = \nabla (\rho)\) in case of mass diffusion = ON. If mass diffusion = OFF, Limiter for all remaining conservative variable will be call right after limiter function for Rho to limit vector U.
  • 2: If mass diffusion = ON, calculate value Rho at all volume and surface Gauss points and then solve \(S_m = \nabla (\rho)\) equation. Results are saved into BR1Vars::massDiffusion::rhoX and BR1Vars::massDiffusion::rhoY.
  • 3: Apply Limiter for vector U.
  • 4: Solve T from U.
  • 5: Solve equation \(S = \mu \nabla (\rho)\).
  • 6: Calculate fluxes (inviscid va viscous) at all faces.
  • 7: Solve NSF
