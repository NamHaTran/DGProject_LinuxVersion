Function gets values all Rho fluxes at all Gauss points on all edges of cell.

Rho flux is calculated by central flux:

\[Flux_\rho = \frac{1}{2}(\rho_+ + \rho_-) \cdot \mathbf{n}\]

Function returns Rho fluxes through 'call by reference'.
